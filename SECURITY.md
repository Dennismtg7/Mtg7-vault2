# Security Policy

## Frontend Game Notice

"African Stone Pit" is a **pure frontend game**. It has no backend, no database, and stores no user data.

## Reporting Issues

If you find a bug or potential exploit (like cheating via browser console), please open a GitHub issue.

There are no critical security vulnerabilities because:
- No personal data is collected
- No server-side code exists
- All game state is local or peer-to-peer
